# 微信小程序扫码工具

一个功能强大的微信小程序扫码工具，支持二维码/条形码扫描和图书条码查询。

## 功能特性

### 1. 普通扫码模式
- 扫描二维码和条形码
- 手动输入条码内容
- 支持网址、ISBN、商品条码等多种类型
- 一键复制扫描结果

### 2. 图书扫码模式
- 扫描图书ISBN条码
- 通过API查询图书详细信息
- 显示书名、作者、出版社、索书号等信息
- 支持自定义图书API接口

### 3. 扫描批次管理
- **扫描批次概念**：每次扫描会话自动形成一个批次
- 历史记录按批次展示，便于管理
- 每个批次显示时间、数量和内容摘要
- 支持批量删除和复制整个批次

### 4. 历史记录功能
- 按批次查看历史扫描记录
- 显示批次时间、扫描数量和预览内容
- 点击进入详情页查看完整记录
- 支持批量操作（删除、复制）

### 5. 设置功能
- 配置图书查询API地址和Key
- 多语言支持（中文/英文）
- 数据导出功能
- 清空历史记录

## 技术架构

- **框架**: 微信小程序原生框架
- **语言**: JavaScript + WXSS + WXML
- **数据存储**: 本地Storage
- **API通信**: wx.request

## 核心功能说明

### 扫描批次
系统采用"扫描批次"概念管理历史记录：
- 每次进入首页开始扫描，自动创建新批次
- 同一次会话内的扫描记录归入同一批次
- 点击"完成扫描"按钮保存当前批次
- 历史记录按批次展示，便于查看和管理

### 数据流
```
扫码/输入 → 添加到当前批次 → 点击完成 → 保存批次 → 历史记录展示
```

## 使用说明

### 1. 普通扫码
1. 打开小程序，选择"普通扫码"模式
2. 点击"点击扫码"按钮或手动输入条码
3. 扫描结果自动添加到当前批次
4. 点击"完成扫描"保存批次
5. 在历史记录页查看和管理

### 2. 图书扫码
1. 切换到"图书扫码"模式
2. 在设置页配置图书API地址和Key
3. 扫描图书ISBN条码
4. 系统自动查询并显示图书信息
5. 保存到当前批次

### 3. API配置说明
在设置页配置以下参数：
- **API地址**: 图书查询接口的URL
- **API Key**: 接口认证密钥（如需要）

支持的API格式：
- GET请求
- 参数: `barcode`（条码）, `apikey`（密钥）
- 返回: JSON或XML格式图书数据

### 4. 历史记录管理
1. 进入"历史"标签页查看所有批次
2. 点击批次查看详情
3. 勾选批次进行批量删除或复制
4. 在详情页查看批次内所有记录

## 安全说明

- API Key存储在本地，不会上传到服务器
- 扫描历史仅保存在本地设备
- 建议定期备份重要数据
