# 微信小程序扫码工具

一个功能强大的微信小程序扫码工具，支持二维码/条形码扫描和图书条码查询。

## 功能特性

### 1. 普通扫码模式
- 扫描二维码和条形码
- 手动输入条码内容
- 支持网址、ISBN、商品条码等多种类型
- 一键复制扫描结果

### 2. 图书扫码模式
- 扫描图书ISBN条码
- 通过API查询图书详细信息
- 显示书名、作者、出版社、索书号等信息
- 支持自定义图书API接口

### 3. 扫描历史管理
- 查看所有扫描记录
- 批量选择和删除
- 批量复制多个结果
- 重复内容自动去重并置顶

### 4. 设置功能
- 配置图书查询API地址和Key
- 多语言支持（中文/英文）
- 数据导出功能
- 清空历史记录

## 技术架构

- **框架**: 微信小程序原生框架
- **语言**: JavaScript + WXSS + WXML
- **数据存储**: 本地Storage
- **API通信**: wx.request

## 使用说明

### 1. 普通扫码
1. 打开小程序，选择"普通扫码"模式
2. 点击"点击扫码"按钮或手动输入条码
3. 查看扫描结果，可点击复制

### 2. 图书扫码
1. 切换到"图书扫码"模式
2. 在设置页配置图书API地址和Key
3. 扫描图书ISBN条码
4. 查看图书详细信息

### 3. API配置说明
在设置页配置以下参数：
- **API地址**: 图书查询接口的URL
- **API Key**: 接口认证密钥（如需要）

支持的API格式：
- GET请求
- 参数: `barcode`（条码）, `apikey`（密钥）
- 返回: JSON或XML格式图书数据

## 安全说明

- API Key存储在本地，不会上传到服务器
- 扫描历史仅保存在本地设备
- 建议定期备份重要数据
